On Time Developer Guide

Introduction

On Time is an application that helps users manage their time. The main page menu displays the calendar, blacklist, and statistics.

Features

  • Calendar
  • Task Scheduling
  • Mute Notifications
  • Website Blocking
  • Personal Statistics

Installation

Download and install our app from the App store or Google Play.

Built With

React Native
